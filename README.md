# LogX - Log Adapter

Unified logging with Golang

## Install

```shell
go get -u github.com/logx-go/logadapter
```

## Development

### Requirement
- Golang >=1.20
- golangci-lint (https://golangci-lint.run/)

### Tests

```shell
go test ./... -race
```

### Lint

```shell
golangci-lint run
```
