# LogX - Log Adapter

Adapter to wrap loggers from Golang log package (https://pkg.go.dev/log)

## Install

```shell
go get -u github.com/logx-go/log-adapter
```

## Development

### Requirement
- Golang >=1.20
- golangci-lint (https://golangci-lint.run/)

### Tests

```shell
go test ./... -race
```

### Lint

```shell
golangci-lint run
```
